# MVP 정의서
## TrendLens - AI 기술 트렌드 뉴스 집약 서비스

**작성일**: 2026-03-26
**버전**: v0.1

---

## MVP 목표

"매일 아침 기술 뉴스 인기 기사를 한국어로 보여주는 웹 페이지"를 동작하게 만든다.

---

## MVP 범위

### ✅ 포함

| # | 기능 | 설명 |
|---|------|------|
| 1 | 뉴스 크롤링 | Reddit, Geeknews, Anthropic Blog 3개 소스 우선 구현 |
| 2 | 인기도 순위 | 댓글 수 + 업보트 기준 정렬 |
| 3 | 한국어 번역 | 제목 + 요약 1줄 번역 (Gemini API) |
| 4 | 오늘의 핵심 이슈 | 상위 10개 기사 기반 AI 요약 3~5줄 |
| 5 | 메인 피드 화면 | 순위 + 뉴스 카드 목록 |
| 6 | 키워드 툴팁 | 기사 키워드 추출 + 호버 설명 |
| 7 | 수동 크롤링 | 설정 페이지에서 "지금 수집" 실행 |
| 8 | 소스 관리 | 설정 페이지에서 크롤링 소스 추가/삭제 |

### ❌ 제외 (추후 구현)

| 기능 | 이유 |
|------|------|
| 로그인/회원가입 | MVP 이후 확장 |
| 개인화 피드 | 로그인 기반 필요 |
| 자동 스케줄링 (07:00) | MVP는 수동 실행으로 대체 |
| EO Planet, Nvidia Blog 크롤링 | 사이트 구조 파악 후 추가 |
| 카테고리 필터 | 단일 피드로 우선 제공 |

## 성공 기준

- 수동으로 크롤링 실행 시 3개 소스에서 기사 수집 완료
- 메인 페이지에서 오늘의 핵심 이슈 + 뉴스 카드 10개 이상 표시
- 기사 키워드에 마우스 올리면 툴팁 표시
- 설정 페이지에서 소스 추가/삭제 동작
